How to Build a Blog on Github Page

这是一篇在github搭博客的简易教程。我在搭博客的过程中，发现虽然可以搜到的教程非常多，但由于软件的版本更迭，许多命令语句已经发生了变化，而大部分教程只提供了语句，并未讲明可从哪里获取最新的对应语句，也未讲明原理，使得当部分平台的相应流程改变后，搭建步骤变得不再可用。
本文会在列出当前可用的博客搭建步骤基础上，写明其原理，和未来版本更迭后可能的新语句查询处。
